Device Management System

Description

A system for managing all company devices, where each product manufactured will be managed through this system. It includes functionalities such as OTA, software management, firmware upgrade, certificate management, device authentication, platform management, and platform management assignment ...

Role as service owner, responsible for building the entire project.
